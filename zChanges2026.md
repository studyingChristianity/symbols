## 1. Substance系の制限

| 変更                | 期待される変化                                                                             |
| ----------------- | ----------------------------------------------------------------------------------- |
| **No cigarettes** | - ニコチン依存解消 → 血圧・心拍安定<br>- 肺機能改善、呼吸が楽になる<br>- 味覚・嗅覚の回復<br>- ドーパミン過剰刺激が減り、自然な快楽に敏感になる |
| **No alcohol**    | - 肝臓負担軽減、脂肪肝リスク減少<br>- 睡眠の質向上（REM睡眠増加）<br>- 不安・抑うつ軽減<br>- 脳の認知機能の維持                 |
| **No cannabis**   | - 集中力・記憶力の回復<br>- 動機付け（ドーパミン反応）が正常化<br>- 不安・疲労感が減る場合が多い                             |
| **No music**      | - 脳の報酬刺激が減る → 他の自然な刺激（散歩・会話・食事）に敏感になる<br>- 注意力向上、心が落ち着く時間増加                         |

> ⚠️ ただし「No music」は人によってはストレス軽減効果もあるので、完全禁止より**制限の管理**が現実的です。

## 1. Substance系制限のリスク・注意点

| 変更                | よくない点・リスク                              | 対策                    |
| ----------------- | -------------------------------------- | --------------------- |
| **No cigarettes** | - ニコチン依存の離脱症状：イライラ、集中力低下、眠気            | 徐々に置き換える、ガムや口さみしさ対策   |
| **No alcohol**    | - 最初は寝つきが悪く感じることがある（アルコールは入眠を助けるが浅い眠り） | リラックス習慣（温かいお茶、深呼吸）で補う |
| **No cannabis**   | - 不安や眠気、食欲の変化が起こる場合                    | 代替リラックス法（瞑想・散歩）で対応    |
| **No music**      | - 気分転換や楽しみが減る → ストレスがたまりやすい            | 完全禁止より、使用時間や種類を管理     |

---

## 2. カフェイン完全制限のリスク

| 変更                                   | よくない点・リスク            | 対策                      |
| ------------------------------------ | -------------------- | ----------------------- |
| **No coffee / energy drinks / coke** | - 依存症状：頭痛、倦怠感、集中力低下  | 徐々に置き換え、最初はお茶で少しカフェイン維持 |
| - 一気にやめると朝の覚醒感が減る                    | 朝のルーチンを水や温かいお茶に置き換える |                         |

---

## 3. 「No sugar」マインドの解除

* **利点はあるが**、甘いものを好きなだけにしてしまうと

  * 血糖値の急上昇 → エネルギーの急降下
  * 体重増加・脂質代謝の悪化
* ポイントは **「適量でOK」** で、制限を完全放棄するのではないこと

---

## 4. 歩く習慣・運動量増加の注意点

* 急に歩きすぎると筋肉痛、関節痛、疲労感
* 足腰の弱い人は**膝・腰の負担**に注意
* 対策：最初は短時間＋頻度少なめ、徐々に増やす

---

## 5. 食事・コレステロール制限

* 極端に制限しすぎると **栄養不足（脂溶性ビタミンや必須脂肪酸）** になり得る
* 対策：バランスを意識し、植物油・魚・ナッツなども摂取

---

## 6. 精神面の注意点

* **刺激を減らす生活は最初は退屈・倦怠感を感じやすい**
  → 「やる気低下」「孤独感」「気分の落ち込み」が出ることもある
* 対策：散歩、自然、読書、短時間の趣味、適度な社交で補う

3. 栄養不足や運動の負担
