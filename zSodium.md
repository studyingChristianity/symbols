# 食塩（主成分：塩化ナトリウム）

## 適量摂取時

1. 体内の水分量を調節する（血液量も含む）
2. 神経伝達をコントロールする
3. 筋肉の収縮をコントロールする
4. 消化作用を助ける
5. 菌の増殖を抑制する効果もある

## 不足

1. 倦怠感
2. 筋肉けいれん
3. 血圧が下がる
4. 脳浮腫を起こし、意識障害や痙攣が生じる。

## 健康的な摂取量

成人男性：1日あたり7.5g未満<br/>
成人女性：1日あたり6.5g未満

## 自然な方法で塩分を補う例

* 味噌汁を1杯（約1gの塩分）加える
* 梅干しや漬物を少量取り入れる
* 塩味のあるナッツやチーズをおやつにする
* 汗を多くかく日には経口補水液（electrolytes）を利用する
