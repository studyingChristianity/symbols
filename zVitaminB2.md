# ビタミンB2（**リボフラビン**）

ビタミンB2は**水溶性ビタミン**の一種で、体内の**エネルギー産生**や**皮膚・粘膜の健康維持**に欠かせない栄養素です。体内でほとんど作れないため、**毎日の食事から摂取**する必要があります。

---

## 主な働き（体への影響）

| 働き       | 具体的な内容                    |
| -------- | ------------------------- |
| エネルギー代謝  | 糖質・脂質・たんぱく質を分解し、エネルギーに変える |
| 皮膚・粘膜の健康 | 口・唇・舌・皮膚の正常な状態を保つ         |
| 成長のサポート  | 体の成長や細胞の新陳代謝を助ける          |
| 抗酸化の補助   | 他の栄養素と協力して、体を酸化ストレスから守る   |

👉 **ポイント**：疲れにくい体づくりや、肌・口元の健康に関係します。

---

## 不足するとどうなる？

| 不足時に起こりやすい症状 | 説明             |
| ------------ | -------------- |
| 口角炎          | 口の端が切れたり、赤くなる  |
| 口内炎・舌炎       | 口の中や舌が痛くなる     |
| 皮膚トラブル       | 肌荒れ、脂っぽいフケなど   |
| 目の疲れ         | 目が充血しやすい、疲れやすい |

※成長期の人や、偏った食事が続くと起こりやすくなります。

---

## 過剰に摂るとどうなる？

ビタミンB2は水溶性なので、**摂りすぎても余分は尿として排出**されます。

| 項目  | 内容                 |
| --- | ------------------ |
| 過剰症 | 基本的に報告はほとんどない      |
| 尿の色 | 一時的に黄色くなることがある（無害） |

👉 通常の食事や一般的なサプリ量で心配はほぼありません。

---

## 多く含まれる食品

| 食品       | 特徴          |
| -------- | ----------- |
| レバー（豚・牛） | 非常に豊富       |
| 牛乳・ヨーグルト | 毎日摂りやすい     |
| 卵        | 調理しやすく吸収も良い |
| 納豆       | 植物性で手軽      |
| うなぎ      | ビタミンB群が豊富   |
| アーモンド    | 間食にも向く      |

---

## 効率よく摂るコツ

| コツ            | 理由         |
| ------------- | ---------- |
| 毎日こまめに摂る      | 体に貯めにくい    |
| 朝食で乳製品を摂る     | 手軽で継続しやすい  |
| ビタミンB群をまとめて摂る | 互いに働きを助け合う |

---

## まとめ

| 項目    | 要点              |
| ----- | --------------- |
| 役割    | エネルギー代謝・皮膚や口の健康 |
| 不足すると | 口内炎、肌荒れ、疲れやすさ   |
| 過剰摂取  | 心配ほぼなし          |
| 摂取源   | レバー、乳製品、卵、納豆など  |
